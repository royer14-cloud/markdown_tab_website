# Interfaz de Markdown Tab

## Barra de iconos superior

La barra superior de Markdown Tab ofrece herramientas rápidas para gestionar y visualizar tus cancioneros. Cada icono cumple una función específica:

 - Nuevo: Limpia el documento actual y añade la estructura básica para comenzar una nueva canción.
 - Importar: Permite importar un archivo con extensión .mdg para continuar editando una canción guardada anteriormente.
 - Guardar: Guarda el documento actual en tu dispositivo.
 - Añadir notas de acordes: (En desarrollo) Añade anotaciones y notas adicionales a los acordes de la canción.
 - Ejecutar: Previsualiza el documento actual en un formato de cancionero, mostrando acordes, tablaturas y estructura.
 - Tema: Cambia entre el modo claro y oscuro para adaptar la interfaz a tus preferencias visuales.
 - Configuración: Abre una ventana con opciones para personalizar el diseño y formato del documento.

Estos iconos facilitan la creación, edición y visualización de cancioneros de forma intuitiva y eficiente
